# portfolio-projects
Some projects I did

Clone the repo:
git clone https://github.com/LemonadeIcedTea/portfolio-projects.git

Install matplotlib on your OS
example: pip install matplotlib
Verify by running this on a Python shell:
import matplotlib.pyplot as plt (if there are no errors, its installed)

Install pandas on your OS
example: pip install pandas
Verify by running this on a Python shell:
import pandas as pd (if there are no errors, its installed)

Install seaborn on your OS
example: pip install seaborn
Verify by running this on a Python shell:
import seaborn as sns (if there are no errors, its installed)

Install stats on your OS
example: pip install scipy
Verify by running this on a Python shell:
from scipy import stats (if there are no errors, its installed)

Install numpy on your OS
example: pip install numpy
Verify by running this on a Python shell:
import numpy as np (if there are no errors, its installed)

Verify if sqlite3 is installed by:
import sqlite3
print(sqlite3.version)

If you encounter ModuleNotFoundError, try reinstalling Python with the "Install pip" and "Add Python to PATH" checked
