This repository contains projects showcasing Python, pandas, matplotlib, seaborn, numpy and SQL. I've attached links leading to each project's README file for more details.

# Projects
-[Online Sports Revenue](portfolio-projects/online_sports_revenue/README.md) - Analyzed Adidas and Nike sales data by merging multiple datasets, categorizing listing prices into quartiles and analyzed product description length relates to customer ratings and reviews.

-[Generating Keywords for Search Campaign](portfolio-projects/generating_keywords_for_search_campaigns/README.md) -Built a keyword generation pipeline for marketing campaigns, exporting results to CSV.

-[Electric Vehicle Charging Trends](portfolio-projects/electric_vehicle_charging_trends/README.md) - Visualized EV charging patterns using pandas and seaborn.

-[Sleep Health](portfolio-projects/sleep_health/README.md) -  Investigated which occupation had the lowest average sleep duration and sleep quality through Python-based data analysis.

-[Avocado Toast Supply Chain Analysis](portfolio-projects/avocado_toast_supply_chain_analysis/README.md) - Examined supply chain data using pandas and custom Python functions to filter the data and return the result.
