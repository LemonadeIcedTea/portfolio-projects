This repository contains projects showcasing Python, pandas, matplotlib, seaborn, numpy and SQL. I've attached links leading to each project's README file for more details.

# Projects
-[Online Sports Revenue](portfolio-projects/online_sports_revenue/README.md) - Analyzed Adidas and Nike sales data by merging multiple datasets, categorizing listing prices into quartiles and analyzed product description length relates to customer ratings and reviews.

-[Generating Keywords for Search Campaign](portfolio-projects/generating_keywords_for_search_campaigns/README.md) -Built a keyword generation pipeline for marketing campaigns, exporting results to CSV.

-[Electric Vehicle Charging Trends](portfolio-projects/electric_vehicle_charging_trends/README.md) - Visualized EV charging patterns using pandas and seaborn.

-[Sleep Health](portfolio-projects/sleep_health/README.md) -  Investigated which occupation had the lowest average sleep duration and sleep quality through Python-based data analysis.

-[Avocado Toast Supply Chain Analysis](portfolio-projects/avocado_toast_supply_chain_analysis/README.md) - Examined supply chain data using pandas and custom Python functions to filter the data and return the result.

-[Debugging Sales Data Workflow](portfolio-projects/debugging_sales_data_workflow/README.md)) - Fixed dataset loading issues and ensured proper initialization of missing tax values.

-[Crime in Los Angeles](portfolio-projects/crime_in_los_angeles/README.md) - Analyzed crime data by identifying the hour with the highest frequency of incidents, determining which are experiences the most nightly crimes and categorizing victims into age groups to show the distribution of crimes committed against them.

[Netflix Movies](portfolio-projects/netflix_movies/README.md) - Filtered and visualized Netflix movie durations from the 1990s to identify the most common runtime, then refined the dataset to include only action films under 90 minutes.

[Covid Workout](portfolio-projects/covid_workout/README.md) - Analyzed and visualized the year when public interest in workouts peaked, identified the types of workouts most popular during the pandemic and today, compared workout interest across three countries, and examined whether Malaysia or the Philippines showed greater interest in home workouts.

[Student's Mental Health](portfolio-projects/students'_mental_health/README.md) - A query consisting the amount of years international students are staying, the number of students and their mental health diagnostic scores.

[Customer Purchase](portfolio-projects/customer_purchase/README.md) - Merge multiple datasets and categorized products into price tiers to compare brand performance and examined whether description length impacts customer ratings and reviews.
