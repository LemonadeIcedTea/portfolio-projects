GeneratingKeywords for Search Campaigns

This project generates at least 60 unique keywords for a search campaign and saves them into a CSV file. The keywords are combinations of product names and promotional words

Output 
- Keywords Saved in a CSV File
  Created two arrays (product and words) and combined them into keywords_list. Converted this list into a dataframe with the columns Ad Group, Keyword, Campaign, and Criterion Type. Assigned fixed values for Campaign (Sem_Sofas) and Criterion Type (Exact). Finally, exported the dataframe as a CSV file into the data/ folder.
